description: NusaUI is a registry-first component system for building product interfaces with Radix UI, Tailwind, and TypeScript.
---

NusaUI is built for teams that want components they can own, inspect, and extend directly in their codebase. Install what you need, tune the implementation, and keep shipping without a runtime dependency layer.

## What NusaUI Gives You

| Area                  | What it means for you                                         |
| --------------------- | ------------------------------------------------------------- |
| **Radix foundations** | Accessible primitives with solid keyboard and focus behavior  |
| **Code ownership**    | Components are copied into your app, not hidden behind a SaaS |
| **Registry workflow** | Install from the CLI or direct registry URLs                  |
| **TypeScript-first**  | Strong typing across components, blocks, and helpers          |
| **Tailwind-ready**    | Styling stays utility-driven and easy to reshape              |
| **Composable design** | Mix, extend, or replace pieces without fighting the system    |

## Working Style

- **Own the source** - every component lands in your repository
- **Keep it accessible** - focus on keyboard flow, labels, and readable feedback
- **Stay lightweight** - no extra runtime wrapper for the basic path
- **Customize early** - adapt the code to your product instead of waiting for a theme API
