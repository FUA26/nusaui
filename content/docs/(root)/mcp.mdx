## What is MCP?

[Model Context Protocol (MCP)](https://ui.shadcn.com/docs/mcp) is an open protocol that enables AI assistants to connect to component registries. With MCP, you can install NusaUI components using natural language.

## Setup

### 1. Configure MCP Server

Add the shadcn MCP server to your project.

<TabsList variant="underline">
  <TabsTrigger value="claude">Claude Code</TabsTrigger>

<TabsPanel value="claude">

Create `.mcp.json` in your project root:

```json title=".mcp.json"
{
  "mcpServers": {
    "shadcn": {
      "command": "npx",
      "args": ["shadcn@latest", "mcp"]
    }
  }
}
```

Restart Claude Code and run `/mcp` to verify the connection.

</TabsPanel>

### 2. Add NusaUI Registry

Configure the NusaUI registry in your `components.json`:

```json title="components.json"
{
  "$schema": "https://shadcn.com/schema.json",
  "registries": {
    "@nusa-ui": "https://nusa-ui.vercel.app/r/{name}.json"
  }
}
```

## Usage

Once configured, ask your AI assistant to install components naturally:

- "Add the button component from @nusa-ui"
- "Install @nusa-ui/code-block"
- "Add a date picker from @nusa-ui"
- "List all available @nusa-ui components"

The AI will use the MCP server to fetch and install components from the NusaUI registry.

## Available Commands

The MCP server supports these operations:

| Command         | Description                                |
| --------------- | ------------------------------------------ |
| List components | Browse all available components and blocks |
| Search          | Find components by name or functionality   |
| Install         | Add components to your project             |
| Get info        | View component dependencies and API        |

## Without MCP

You can always install components directly without MCP:

```bash
npx shadcn@latest add @nusa-ui/button
```

Or using the full URL:

```bash
npx shadcn@latest add https://nusa-ui.vercel.app/r/button.json
```
