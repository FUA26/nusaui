## API Reference

### CopyButton

Extends the `Button` component with copy functionality.

| Prop      | Type         | Default     | Description                    |
| --------- | ------------ | ----------- | ------------------------------ |
| `value`   | `string`     | -           | The text to copy to clipboard  |
| `onCopy`  | `() => void` | -           | Called after successful copy   |
| `timeout` | `number`     | `2000`      | Duration of success state (ms) |
| `variant` | `string`     | `"outline"` | Button variant                 |
| `size`    | `string`     | `"icon"`    | Button size                    |

All other `Button` props are supported.
