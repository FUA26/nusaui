## Usage

```tsx
import {
  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableHead,
  TableHeader,
  TableRow,
} from "@/components/ui/table"
```

```tsx
<Table>
  <TableCaption>A list of items.</TableCaption>
  <TableHeader>
    <TableRow>
      <TableHead>Name</TableHead>
      <TableHead>Status</TableHead>
    </TableRow>
  </TableHeader>
  <TableBody>
    <TableRow>
      <TableCell>Item 1</TableCell>
      <TableCell>Active</TableCell>
    </TableRow>
  </TableBody>
</Table>
```

## API Reference

### Table

| Component      | Description              |
| -------------- | ------------------------ |
| `Table`        | The root table element   |
| `TableHeader`  | The table header section |
| `TableBody`    | The table body section   |
| `TableFooter`  | The table footer section |
| `TableRow`     | A table row              |
| `TableHead`    | A header cell            |
| `TableCell`    | A body cell              |
| `TableCaption` | A caption for the table  |
