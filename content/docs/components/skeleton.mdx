## API Reference

### Skeleton

The `Skeleton` component accepts all standard `div` props plus Tailwind classes for sizing and styling.

| Prop        | Type     | Default | Description            |
| ----------- | -------- | ------- | ---------------------- |
| `className` | `string` | -       | Additional CSS classes |
