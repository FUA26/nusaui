## Usage

```tsx
import {
  Pagination,
  PaginationContent,
  PaginationItem,
  PaginationLink,
  PaginationNext,
  PaginationPrevious,
} from "@/components/ui/pagination"
```

```tsx
<Pagination>
  <PaginationContent>
    <PaginationItem>
      <PaginationPrevious href="#" />
    </PaginationItem>
    <PaginationItem>
      <PaginationLink href="#">1</PaginationLink>
    </PaginationItem>
    <PaginationItem>
      <PaginationNext href="#" />
    </PaginationItem>
  </PaginationContent>
</Pagination>
```

## API Reference

### Pagination

| Component            | Description                    |
| -------------------- | ------------------------------ |
| `Pagination`         | The root component             |
| `PaginationContent`  | Container for pagination items |
| `PaginationItem`     | A single pagination item       |
| `PaginationLink`     | A page number link             |
| `PaginationPrevious` | Previous page link             |
| `PaginationNext`     | Next page link                 |
| `PaginationEllipsis` | Ellipsis for skipped pages     |

### PaginationLink Props

| Prop       | Type      | Default | Description                |
| ---------- | --------- | ------- | -------------------------- |
| `href`     | `string`  | -       | The URL the link points to |
| `isActive` | `boolean` | `false` | Mark as current page       |
