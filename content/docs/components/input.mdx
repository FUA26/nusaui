## Props and parts

### Input

All standard HTML input attributes are supported.

| Prop          | Type      | Default  | Description       |
| ------------- | --------- | -------- | ----------------- |
| `type`        | `string`  | `"text"` | The input type    |
| `placeholder` | `string`  | -        | Placeholder text  |
| `disabled`    | `boolean` | `false`  | Disable the input |
