## Usage

```tsx
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSlot,
} from "@/components/ui/input-otp"
```

```tsx
<InputOTP maxLength={6}>
  <InputOTPGroup>
    <InputOTPSlot index={0} />
    <InputOTPSlot index={1} />
    <InputOTPSlot index={2} />
    <InputOTPSlot index={3} />
    <InputOTPSlot index={4} />
    <InputOTPSlot index={5} />
  </InputOTPGroup>
</InputOTP>
```

## API Reference

### InputOTP

The root component that manages the OTP input state.

| Prop        | Type                      | Default              | Description                   |
| ----------- | ------------------------- | -------------------- | ----------------------------- |
| `maxLength` | `number`                  | -                    | Maximum number of characters  |
| `value`     | `string`                  | -                    | Controlled value              |
| `onChange`  | `(value: string) => void` | -                    | Called when the value changes |
| `pattern`   | `string \| RegExp`        | `REGEXP_ONLY_DIGITS` | Pattern to validate input     |
| `disabled`  | `boolean`                 | `false`              | Disable the input             |
| `autoFocus` | `boolean`                 | `false`              | Auto focus on mount           |

### InputOTPGroup

Groups slots together visually.

### InputOTPSlot

Individual character slot.

| Prop    | Type     | Default | Description                     |
| ------- | -------- | ------- | ------------------------------- |
| `index` | `number` | -       | The index of the slot (0-based) |

### InputOTPSeparator

Visual separator between groups.
