## Installation

```bash
npx shadcn@latest add https://nusa-ui.vercel.app/r/login-form.json
```

## Usage

This block is a **complete login page** ready to use. Just import and render:

```tsx
import { LoginForm } from "@/components/blocks/login-form"

export default function LoginPage() {
  return <LoginForm />
}
```

## Features

- Full page layout with centered card
- Logo with link to homepage
- Email field with Mail icon
- Password field with Lock icon and show/hide toggle
- Remember me checkbox
- Forgot password link
- Zod validation
- TanStack Form for form state management
- Submit button with spinner
- Sign up link
- Footer with copyright

## Customization

- Replace the logo with your own
- Modify the validation schema in `loginSchema`
- Integrate with your authentication provider in `onSubmit`
- Adjust default remember me value in `defaultValues`
- Change the links (`/`, `/forgot-password`, `/sign-up`) to match your routing
